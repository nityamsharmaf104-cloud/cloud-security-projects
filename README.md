# Cloud Security Projects

Hands-on AWS cloud security and DevOps labs built during my summer break.
Each project covers a different area of cloud security or DevOps engineering.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [IAM Least Privilege Lab](iam-least-privilege-lab/) | IAM users, groups, policies and access validation |
| 2 | [S3 Security Lab](s3-security-lab/) | Bucket misconfiguration detection and remediation |
| 3 | [IAM Access Analyzer Lab](iam-access-analyzer-lab/) | Automated external access detection and remediation |
| 4 | [Prowler Security Audit](prowler-security-audit/) | Automated AWS account security scan — 301 checks, 71 findings |
| 5 | [CI/CD Pipeline Lab](ci-cd-pipeline-lab/) | Continuous deployment pipeline — GitHub to S3 via CodePipeline |
